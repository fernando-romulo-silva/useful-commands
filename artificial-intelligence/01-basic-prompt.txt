The following delimiters are commonly used:

1) """ (three double quotation marks):
To indicate a text that should not be processed or interpreted.
Ex:

Give a title to the text below:

Text:

"""
Python is a high-level, multi-paradigm, general-purpose language that supports object-oriented, imperative, functional, and procedural paradigms. It has dynamic typing, and one of its main characteristics is that it allows for easy code readability and requires fewer lines of code compared to the same program in other languages.
"""

Title:

2) ```(three backticks): 
The three backticks are used to indicate that the content between them is treated as a block of code. 
Ex:

Explain the code below:

```
for i in range(5):
   print(i)
```

3) _____ (underlines): 
Underlines can be used to generate a form-format result. 
This is useful if you want to automate the output of a prompt and don't want the result to be in code, only in text.
Ex:

I'm launching a Mother's Day campaign for a beauty kit. 
The YouTube video needs a title, description, thumbnail idea, and thumbnail text.
Suggest a duration in minutes and which channels I can partner with to promote the campaign.
Also, describe its strengths. Please submit the result in the format below:

Title: _____
Description: _____
Thumbnail idea: _____
Thumbnail text: _____
Duration: _____
Channels: _____, _____,_____
Strengths: _____,_____,_____


4) Use quotations: 
Use quotation marks to quote relevant passages of text in your prompt, especially if you are asking a question based on a quote from a text.

What is the book "Storytelling with Data" by author Cole Nussbaumer Knaflic about?

5) Be clear and specific: 
When writing a prompt, it's important to be clear and specific about what you want ChatGPT to do. 
This helps the model understand exactly what you're asking and generate a more precise response.
Therefore, it's important to avoid using vague or ambiguous terms that could confuse the model.

For example, instead of writing "Give me information about Java," try writing "What is the syntax of the Java programming language?"


6) Provide context: 
Providing additional information or relevant context to ChatGPT can help the model better understand the question and generate a more accurate response. 
If you are asking a question about a specific topic, you can provide some basic information about that topic in the prompt.

For example, if you are asking a question about an error in Python code, provide some basic details about what that code is.

7) Avoid complex questions:

Avoid asking complex questions or questions that require detailed answers. 
ChatGPT works best when it receives simple and direct questions. 
Therefore, try breaking down complex questions into smaller, simpler ones.


8) Let's think step-by-step

This year the company's revenue grew by 20%, but profit (EBITDA) fell by 10%. What happened?
